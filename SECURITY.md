# セキュリティポリシー / Security Policy

## サポート対象 / Supported versions

本リポジトリはテンプレートのため、最新の `main` ブランチのみをサポート対象とします。
As this repository is a template, only the latest `main` branch is supported.

## 脆弱性の報告 / Reporting a vulnerability

脆弱性を見つけた場合は、**公開 Issue を作成せず**、以下のいずれかで非公開に報告してください。
If you find a vulnerability, please report it privately — **do not open a public issue** — via one of:

- GitHub の **Security → Report a vulnerability**（Private vulnerability reporting）/ GitHub's **Security → Report a vulnerability** (Private vulnerability reporting)
- メール / Email: security@itouh.com

### 報告に含めてほしい情報 / What to include

- 影響範囲と再現手順 / Impact and steps to reproduce
- 該当バージョン・コミット / Affected version or commit
- 可能なら PoC / Proof of concept if possible

## 対応の流れ / Process

- 受領後、合理的な期間内に確認の返信をします。/ We will acknowledge your report within a reasonable timeframe.
- 修正後に公開し、希望があれば報告者をクレジットします。/ We will disclose after a fix and credit reporters who wish to be named.
