claude-fullstack-starter 概要

Vue 3 + Vite + TypeScript × FastAPI / VSCode Dev Container — 開発プロセスと品質ゲートの全体像

技術スタック

  • Backend: FastAPI / Python 3.14 / Pydantic / pytest / ruff
  • Frontend: Vue 3 (Composition API + script setup) / Vite / TS
  • 環境: VSCode Dev Container (Ubuntu)
  • CI: GitHub Actions (ruff / pytest / eslint / type-check)

リポジトリ構造

  • backend/app: api(ルーター集約) / config / main
  • frontend/src: services(API集約) / App.vue / main.ts
  • .github/workflows: CI・保護・sandbox
  • .claude/skills + CLAUDE.md: 規約とスキル
  • docs/: 構造・プロセスの図解資料

リクエストの流れ

  • ブラウザ → Vite dev (5173)
  • /api/* を Vite proxy で backend (8000) へ
  • FastAPI: /api prefix + ルーター
  • フロントは必ず services/ 経由で通信

ブランチ戦略

  • main: リリース対象 (Ruleset 保護)
  • dev: 統合ブランチ
  • feat/* fix/* docs/* ci/*: 作業
  • sandbox/main: 検証の基点 (main ミラー)
  • sandbox/<名前>: 使い捨て検証

CI ワークフロー

  • ci.yml: main の品質チェック
  • sandbox-ci.yml: sandbox/** で CI 実行
  • block-sandbox-pr.yml: sandbox/* → main を自動クローズ
  • ジョブ: backend / frontend

ブランチ保護 (Rulesets)

  • PR 必須 + 必須チェック backend / frontend
  • strict (最新取込) / force push・削除禁止
  • bypass なし = admin もマージ不可
  • CI 緑のときだけマージ可能
