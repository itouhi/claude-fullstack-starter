# M8 closing-tax — ドキュメント索引

> 個人事業主向け会計システム モジュール M8: 決算・確定申告・消費税

## 概要

決算整理仕訳の生成(F-801)・青色申告決算書の集約出力(F-802)・申告データ出力(F-803)・消費税簡易課税集計(F-804)を担う Layer 3 モジュール。M6(固定資産)・M7(帳簿集計)の完成を前提とする。
