## 主要な設計判断サマリ

- **仕訳方式**: 直接法 (借方: 減価償却費609 / 貸方: 工具器具備品150)
- **計算式**: 定額法 — `floor(取得価額 / 耐用年数)` (円未満切り捨て)
- **期中月割**: `floor(年間償却費 × (12 − 取得月 + 1) / 12)`
- **備忘価額**: 最終年度は1円残し (`期首残高 − 1 円`)
- **少額特例**: 取得価額 < 30万円 → 取得年度に `取得価額 − 1 円` を即時償却
- **家事按分**: `減価償却費609 = 年間償却費 × 事業専用割合`、差額を事業主貸410へ
